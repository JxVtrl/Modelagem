Como Funciona?

Esse programa foi feito para explicar os processos de adição e subtração de forma visual.

Primeiramente será necessário escolher a operação desejada. Você pode escolher entre Soma e Subtração. Assim que a operação for escolhida, será preciso escolher a base numérica para o cálculo.

Note que os processos de soma e subtração comuns são feitos em base 10, mas o programa suporta bases de 2 (binária) até 36, aceitando, portanto, algarismos de 0 até Z, contanto que dentro dos limites da base escolhida.

Com a operação selecionada, novos campos abrirão, com espaços para os valores que você deseja somar ou subtrair na base escolhida.

Quando a base e os valores forem escolhidos, basta clicar no botão "Calcular" que a resposta será exibida logo em seguida.
